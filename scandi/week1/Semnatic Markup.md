# Semnatic Markup

```Html
<!-- non-semnatic html-->
<div id="footer">
	<p>(copyright)text</p>
</div>
```
```html
<!-- semnatic html -->
<footer>
	<p>(copyright)text</p>
</footer>
```
---
### Categorize
1. Generic tags 
	1. <span>
	2. <div>
2. Content tags
	1. <h1>
	2. <p>
	3. <li>
	4. <ul>
	5. <img>
3. Structure tags
	1. <header>
	2. <main>
	3. <footer>
	4. <article>
	5. <section>
