# Obsidian  Hotkeys
________________________________________________________________________
### 1. Open Quick Switcher

```hotkey
CTRL + O 
```

### 2. Open New Tab
```hotkey
CTRL + T
```

### 3. Close Tab
```hotkey
CTRL + W
```

### 4. Select Command

```hotkey
CTRL + W
```
- `date` - insert current date
- `time` - insert current time 
- `filesnav`  - where is file

### 5. Make bold text

```hotkey
CTRL + B
```
